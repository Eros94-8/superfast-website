SuperFast API Documentation

Welcome to the official documentation for the SuperFast Prime Checking API.

Base URL

https://superfast-api.onrender.com

Authentication

All requests require your API key:

x-api-key: YOUR_API_KEY

POST /prime/pro

Check if a number is prime.

curl -X POST "https://superfast-api.onrender.com/prime/pro" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"n": 999999937}'

GET /usage

Check your daily usage.

curl -X GET "https://superfast-api.onrender.com/usage" \
  -H "x-api-key: YOUR_API_KEY"

Error Responses

{ "error": "Invalid API key" }
{ "error": "Daily limit exceeded" }
{ "error": "Missing parameter n" }
