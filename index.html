API Reference

POST /prime/pro

Check if a number is prime.

curl -X POST "https://superfast-api.onrender.com/prime/pro" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"n": 999999937}'

GET /usage

Check your daily usage and plan.

curl -X GET "https://superfast-api.onrender.com/usage" \
  -H "x-api-key: YOUR_API_KEY"
